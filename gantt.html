Setup

In the block below, we make a dataframe with the following columns: project_name, start_date, end_date, link, category

Making the Plot

Code 
gantt_df <- gantt_df |>
  dplyr::arrange(
    dplyr::desc(start_date)
  )

plot_gantt(gantt_df)
